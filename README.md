# trackWay

###  Movie and series tracking app. Using themoviedb api.

##### Devolopment
npm run start-server

##### Production
npm run build && npm start


##### Requirement
- node : 12.18.1
- npm i -g cross-env

##### Package versions
- react (16.12.0)
- redux (4.0.5) - react-redux (7.1.1)
- sass (4)
- webpack (4.41.5)
